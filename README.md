# Student Score Analysis

## Overview

This project uses **Python and pandas** to analyze student performance data.

The goal is to practice data handling techniques such as grouping, counting, averaging, sorting, and creating new data columns using pandas methods.

## Dataset

The dataset includes:

* Student names
* Course enrollment
* Exam scores
* Hours studied
* Pass/fail status

## Analysis Performed

The program answers the following questions:

1. Count students in each course
2. Calculate average score by course
3. Find the top 3 students by score
4. Compare average study hours between passed and failed students
5. Add letter grades based on scores
6. Show grade distribution by course

Run:


This project demonstrates basic data analysis skills used in data science, automation, and AI workflows.
